# 🌐 Руководство по закрузке AI-Ассистента 24/7 (Без ПК)

Чтобы ваш AI-ассистент **Nova AI** работал непрерывно 24 часа в сутки 7 дней в неделю и был доступен с вашего телефона, планшета или любого компьютера без необходимости держать включенным домашний ПК, разверните его на бесплатной облачной платформе.

---

## 🚀 Способ 1: Render.com (100% Бесплатно)

**Render** — надежный облачный хостинг для Node.js приложений с автоматическим SSL (HTTPS).

### Шаги установки:
1. Зарегистрируйтесь на бесплатном сайте [Render.com](https://render.com).
2. Выгрузите папку этого проекта на свой аккаунт **GitHub** или **GitLab**.
3. На панели управления Render нажмите **New +** -> **Web Service**.
4. Подключите ваш репозиторий с проектом.
5. Заполните параметры сборки:
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
6. В разделе **Environment Variables** (Переменные окружения) добавьте свой `GEMINI_API_KEY` или `OPENAI_API_KEY` (если есть).
7. Нажмите **Create Web Service**.

🎉 **Готово!** Через 1-2 минуты Render выдаст вам постоянную ссылку вида `https://nova-ai-assistant.onrender.com`, которая работает 24/7 по всему миру!

## ⚡ Способ 2: Railway.app или Vercel

### Railway:
1. Перейдите на [Railway.app](https://railway.app) и авторизуйтесь через GitHub.
2. Нажмите **New Project** -> **Deploy from GitHub repo**.
3. Выберите репозиторий `uz ai bot`. Railway автоматически определит `server.js` и запустит сервер за 30 секунд!

## 🖥️ Способ 3: На собственном VPS сервере (Linux / Ubuntu)

Если у вас есть недорогой VPS сервер (Ubuntu 20.04/22.04):

1. Скопируйте файлы на сервер:
   ```bash
   git clone <url-вашего-репозитория>
   cd uz-ai-bot
   npm install
   ```

2. Запустите фоновый менеджер процессов **PM2**, чтобы приложение никогда не выключалось:
   ```bash
   npm install -g pm2
   pm2 start server.js --name "nova-ai"
   pm2 save
   pm2 startup
   ```

теперь ваш сервер будет автоматически перезапускаться даже при перезагрузке самого сервера!

## 🛠️ Переменные окружения (`.env`)

Вы можете настроить ключи в облаке или локально в файле `.env`:
- `TELEGRAM_BOT_TOKEN` — Токен бота из Телеграма от `@BotFather`
- `GEMINI_API_KEY` — Ключ Google Gemini API
- `OPENAI_API_KEY` — Ключ OpenAI API
- `PORT` — Порт (по умолчанию 3000)

### 📲 Как получить токен для Telegram Бота за 1 минуту:
1. Откройте Telegram и найдите официального бота **[@BotFather](https://t.me/BotFather)**.
2. Отправьте ему команду `/newbot`.
3. Введите название бота и юзернейм (например, `MyNovaAiBot`).
4. Скопируйте полученный токен (вида `7123456789:AAF...`) и вставьте в файл `.env`:
   `TELEGRAM_BOT_TOKEN=7123456789:AAF...`
